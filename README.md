# Vulnerabilitat residencial i energètica de les llars a Espanya

Aquest repositori conté el codi i les dades preparades per al projecte de visualització de dades de la pràctica final de l’assignatura.

La visualització analitza la vulnerabilitat residencial i energètica de les llars a Espanya a partir de l’Enquesta de Condicions de Vida 2025 de l’INE.

## Visualització final

La visualització final s’ha creat amb Flourish i està disponible públicament en aquest enllaç:

https://public.flourish.studio/story/3699218/

## Font de dades

Les dades originals provenen de les microdades de l’Enquesta de Condicions de Vida 2025 de l’Institut Nacional d’Estadística.

Font: INE - Encuesta de Condiciones de Vida 2025
https://www.ine.es/dyngs/INEbase/es/operacion.htm?c=Estadistica_C&cid=1254736176807&menu=resultados&idp=1254735976608

## Fitxers principals

* `preparar_dades.py`: selecciona i recodifica les variables principals.
* `explorar_dades.py`: fa una exploració inicial de les dades.
* `crear_taules_visualitzacio.py`: crea les taules agregades utilitzades a Flourish.
* `ecv_visualitzacio_final.csv`: dataset preparat.
* `taules_visualitzacio/`: fitxers CSV utilitzats per crear les visualitzacions.
